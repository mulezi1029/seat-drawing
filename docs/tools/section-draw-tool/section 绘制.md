基于 Seats.io 官方文档和交互设计，绘制 Section（区域）的完整交互流程及 UI 状态变化如下：
一、准备阶段：工具选择
1. 点击 Section 工具
左侧工具栏：Section 工具（S 图标）高亮显示，表示已选中
变体菜单展开：工具右侧显示 > 箭头，悬浮或点击后出现两个选项：
矩形图标：快速拖拽绘制矩形区域
多边形图标：逐点绘制任意形状（含曲线）

二、绘制阶段：多边形模式（最常用）
2. 开始绘制（首次点击）
画布状态：鼠标变为十字准星（Crosshair）+ 节点预览
UI 反馈：
在点击位置出现蓝色小方块（第一个节点）
从鼠标位置延伸出一条跟随线（Rubber Band），实时预览下一条边
底部状态栏显示：
"Section | Click insert a node | Right Click remove last node | Shift snap to 45° angle | Alt disable snapping"

3. 添加中间节点
视觉反馈：
每点击一次，新增一个蓝色方块节点
节点之间以蓝色实线连接
最后一个节点与鼠标之间保持虚线/跟随线
曲线预览：如果是宽角度（>45°），线段会实时显示为曲线弧度（贝塞尔曲线预览）
4. 特殊交互：回到起点闭合
当鼠标靠近第一个节点时：
吸附效果：第一个节点放大高亮（通常变为橙色或绿色）
闭合提示：跟随线显示为闭合预览（即将连接起点）
点击结果：
路径自动闭合，形成封闭多边形
区域填充为浅灰色半透明（表示有效区域）
绘制模式自动退出
三、完成阶段：对象选中状态
5. 刚完成绘制的 UI 状态
一旦闭合（点击起点或双击），Section 进入选中编辑模式：
边框显示：
蓝色细线边框：标识当前选中对象
四角蓝色方块：变换控制点（用于缩放）
顶部中心圆点：旋转控制点（用于旋转角度）
内部状态：
显示默认文本 "Section"（标签）
填充色为浅灰色（可在属性面板修改）
6. 右侧属性面板（自动展开）

四、后期编辑：节点精修
7. 使用 Node Tool 调整形状
切换工具：左侧工具栏选择 Node Tool（节点工具）
UI 变化：
Section 边框显示为可编辑路径（带多个蓝色节点）
可拖拽节点移动位置
可点击边线添加新节点
线段实时更新曲率
8. 平滑度调节
拖动 Smoothing 滑块（0% → 100%）：
0%：多边形硬边（折线）
50%：轻微圆角
100%：最大平滑（完美曲线）
五、进入区域内部（双层架构）
9. 双击进入 Section
过渡动画：画布平滑缩放到 Section 内部（视口自适应）
UI 变化：
顶部标题变为 "Exit section" 按钮（带返回箭头）
左侧工具栏切换为座位绘制工具（排座位、桌子、GA 等）
Section 边界变为裁剪边界（超出部分不显示）
