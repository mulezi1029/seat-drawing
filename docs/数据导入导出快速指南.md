# 数据导入导出快速指南

## 5 分钟快速上手

### 导出数据

1. **点击导出按钮**
   - 位置：顶部工具栏右侧
   - 图标：下载图标 (↓)

2. **自动下载文件**
   - 文件名：`{场馆名称}-{时间戳}.json`
   - 位置：浏览器默认下载目录

3. **完成**
   - 数据已保存到本地

### 导入数据

1. **点击导入按钮**
   - 位置：顶部工具栏右侧
   - 图标：上传图标 (↑)

2. **选择文件**
   - 选择之前导出的 JSON 文件

3. **确认导入**
   - 查看数据摘要
   - 点击"确定"

4. **完成**
   - 数据已恢复

## 常见使用场景

### 场景 1：每日备份

```
每天工作结束时：
1. 点击导出按钮
2. 文件自动保存到下载目录
3. 定期整理到云盘
```

### 场景 2：切换设备

```
电脑 A:
1. 导出数据
2. 上传到云盘

电脑 B:
1. 从云盘下载
2. 导入数据
3. 继续工作
```

### 场景 3：版本管理

```
重要修改前：
1. 导出当前版本（backup-v1.json）
2. 进行修改
3. 如需回退，导入 backup-v1.json
```

## 注意事项

⚠️ **导入会替换当前所有数据**
- 导入前建议先导出备份

✅ **支持的文件格式**
- 仅支持 JSON 格式
- 必须是本系统导出的文件

📦 **文件大小**
- 包含背景图的文件可能较大
- 建议使用压缩工具

## 故障排除

### 问题：导入失败

**解决方法**：
1. 检查文件格式是否为 JSON
2. 确认文件未损坏
3. 查看浏览器控制台错误信息

### 问题：背景图不显示

**解决方法**：
1. 检查网络连接（HTTP URL）
2. 重新上传背景图

### 问题：座位位置不对

**解决方法**：
1. 确认文件版本兼容
2. 尝试重新导入
