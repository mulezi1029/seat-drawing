# 座位绘制功能验证清单

## ✅ 核心功能验证

### 1. 单排座位绘制（seats.io 风格）

- [ ] **拖拽绘制**
  - [ ] 按 `1` 键切换到单排座位工具
  - [ ] 在画布上按住鼠标拖拽
  - [ ] 看到蓝色正方形座位预览
  - [ ] 看到蓝色虚线连接起点和终点
  - [ ] 看到黑色方块显示座位数量
  - [ ] 释放鼠标完成绘制

- [ ] **座位自动旋转**
  - [ ] 水平拖拽 → 座位水平排列
  - [ ] 垂直拖拽 → 座位垂直排列
  - [ ] 斜向拖拽 → 座位斜向排列

- [ ] **数量自动计算**
  - [ ] 短距离拖拽 → 少量座位（如 2-3 个）
  - [ ] 长距离拖拽 → 多量座位（如 10+ 个）

### 2. 矩阵座位绘制

- [ ] **拖拽绘制**
  - [ ] 按 `2` 键切换到矩阵座位工具
  - [ ] 拖拽矩形区域
  - [ ] 看到虚线矩形框预览
  - [ ] 看到行列数和总座位数标签
  - [ ] 释放鼠标完成绘制

### 3. ESC 键功能

- [ ] **取消绘制**
  - [ ] 按 `1` 开始拖拽绘制
  - [ ] 拖拽过程中按 `ESC`
  - [ ] 绘制被取消，无座位创建

- [ ] **切换工具**
  - [ ] 按 `1` 或 `2` 切换到绘制工具
  - [ ] 按 `ESC` 键
  - [ ] 自动切换回选择工具

### 4. 座位组选择

- [ ] **单击选中整组**
  - [ ] 绘制一排座位（如 5 个）
  - [ ] 按 `ESC` 切换到选择工具
  - [ ] 单击任意一个座位
  - [ ] 整排 5 个座位全部被选中（蓝色边框）

- [ ] **Ctrl+单击单个座位**
  - [ ] 按住 Ctrl 键
  - [ ] 单击座位
  - [ ] 只选中单个座位

### 5. 座位删除

- [ ] **删除整组**
  - [ ] 单击选中一组座位
  - [ ] 按 `Delete` 或 `Backspace` 键
  - [ ] 整组座位被删除

- [ ] **批量删除**
  - [ ] 框选多个座位
  - [ ] 按 `Delete` 键
  - [ ] 所有选中座位被删除

### 6. 座位移动

- [ ] **移动整组**
  - [ ] 单击选中一组座位
  - [ ] 拖拽移动
  - [ ] 看到半透明预览
  - [ ] 释放鼠标完成移动

- [ ] **批量移动**
  - [ ] 框选多个座位
  - [ ] 拖拽移动
  - [ ] 所有座位一起移动

### 7. 座位属性编辑

- [ ] **单个座位属性**
  - [ ] 选中单个座位
  - [ ] 右侧面板显示属性
  - [ ] 修改行号 → 立即生效
  - [ ] 修改座位号 → 立即生效

- [ ] **批量属性编辑**
  - [ ] 选中多个座位
  - [ ] 修改类型为 "VIP"
  - [ ] 所有座位变为金色
  - [ ] 修改角度为 45°
  - [ ] 所有座位旋转 45°

### 8. 画布交互

- [ ] **缩放**
  - [ ] Ctrl+滚轮向上 → 放大
  - [ ] Ctrl+滚轮向下 → 缩小
  - [ ] 以鼠标位置为中心缩放

- [ ] **平移**
  - [ ] 按住 Space 键
  - [ ] 拖拽画布平移
  - [ ] 释放 Space 键恢复工具

### 9. 保存功能

- [ ] **保存座位数据**
  - [ ] 绘制若干座位
  - [ ] 点击顶部"保存"按钮
  - [ ] 返回主画布
  - [ ] 座位数据保存到区域
